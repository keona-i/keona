2. Semester - Full Stack Development

A Game to understand the Full Stack

The task was to code a Full Stack project. The backend schould be connect to the frontend, thereby we had to embedded at least one LED strip. As project I choose the Game Captain's mistress/Connect Four.



Prototyping

The Ardunio Uno drive with javaScript and controls the LED's. In the Code we defined the individuals pixels, with three factors - 0,1,2. That means the pixel can be free or the chip of the player 1 take the pixel or player 2 take it. One of the player shall click the button in the interface, so the chip drops as a LED down. Through the functions drop and applyGravit in the Code, we can simulated the gravitation, like in the real game. The players have different colors, player 1 has the color red and palyer 2 has the color green. With the reset button they can restart the game and play it all over again.





Full Stack

In the frontend I worked with Vue and Bootstrap, so I can create a interface fast and without Css knowledge. The frontend is connect with the backend via socket, that is actual the reason you can click the button and the LED changes in the playing surface. That implies the data transferred in real-time in socket-io, so the communcation between interface and server. In the backend all the funcation I worte in JavaScript, which is connect with the Arduino Uno, where also the LED is pin to.





Interface

The interface can be open on Website or on the Smartphone with a QR-code.
